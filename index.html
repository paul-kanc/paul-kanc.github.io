Pong


Type: Single / Multiplayer Game

Language Used: Processing



Description


I started the game by creating a moving ball that would collide with the edge of the screen. This then evolved to have colision with controllable paddles and menu screens. The original design for this game was for it to be multiplayer only. After finding it relatively simple to make, I decided to challenge myself and create a single player mode where the right paddle would move on its own. I made the paddle track the y co-ordinate of the ball and used the 'random' function to create an error margin for the lower levels.

Forest Jumper


Type: Single Player Platform Game

Language Used: Java



Description


I started by importing a physics engine created by my university. Once I understood the basic features such as collision to walk on the floor and collect coins, I created a version of the game consisting of a single level that would end after all coins were collected. The game then progressed with the addition of more levels and villains that would remove lives when collided with. The final touches included instructions, the ability to pause the game, a power-up that would allow the villains to be destroyed, sound and a GUI. To further improve the game, I added 3 difficulties that would change the speed of the villains, a low gravity mode that would reduce the variable controlling gravity and a high score table that would read and write from a txt file.

Calculator


Type: Basic Calculator

Language Used: Python



Description


I used the Tkinter module to design the GUI and used partial functions to give the buttons their functionality. The functions that the calculator can perform include addition, subtraction, division, multiplication, negation, bracketing and powers. Before inserting the relevant information into an expression, checks are made to ensure that what is being inserted will result in a valid expression (e.g. 3+-/8 is not valid). The calculator will not allow the user to create an invalid expression.

Multisweeper


Type: Single / Multiplayer Puzzle Game

Language Used: Java



Description


Multisweeper is a multiplayer and single player minesweeper game built using JavaFX for visuals, MySQL for database functionality, and socket programming for the peer-to-peer multiplayer functionality. The aim of the project was to create a multiplayer version of minesweeper where 2 players can compete to see who would complete the board first. The aim for the future is to transfer the project to the web, no longer use peer-to-peer for multiplayer and create additional features to improve gameplay such as achievments.
