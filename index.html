Pong


Type: Single / Multiplayer Game

Language Used: Processing



Description


I started the game by creating a moving ball that would collide with the edge of the screen. This then evolved to have colision with controllable paddles and menu screens. The original design for this game was for it to be multiplayer only. After finding it relatively simple to make, I decided to challenge myself and create a single player mode where the right paddle would move on its own. I made the paddle track the y co-ordinate of the ball and used the 'random' function to create an error margin for the lower levels.
